## headコマンドをつくる

使用例

`head -n 5 args.c`
`cat args.c | head -n 5`

要件

- 先頭の数行を出力する  
- 引数を渡さなかった場合、標準入力から読み込む

#### atoi(3), atol(3)

整数表現を含む文字列を int や long に変換する

#### オプション（マイナスで始まるコマンドライン引数）の慣習

- パラメータを取らないものは1つにまとめてもOK（`ls -als` とか）
- パラメータを取るものはオプションとパラメータをくっつけてOK（`head -n5`とか）
- マイナスを二つつけるものは**ロングオプション**と呼ばれる

#### getopt(3)

オプション解析API

#### getopt_long(3)

ロングオプション解析API
